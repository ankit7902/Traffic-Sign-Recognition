📖 Documentation

🔹 Overview

This project enables traffic sign recognition using an AI-powered model. Users can upload an image or use a live webcam to identify signs.

📌 Features

  • Upload an image for traffic sign detection
  • Live webcam sign recognition
  • Dark mode support
  • Drag & Drop image upload
  • Responsive UI for mobile & desktop

🛠️ How to Use

  1. Click "Upload Traffic Sign" to select an image.
  2. Or, click "Start Webcam" to capture a live image.
  3. Press "Predict Sign" to identify the traffic sign.
  4. The system will display the recognized traffic sign.
